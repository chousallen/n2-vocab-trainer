# JLPT N2 Vocabulary Trainer

Static flashcard trainer for JLPT N2 vocabulary study.

## Features

- Chapter selection
- All/random/weak-card sessions
- Chinese translations
- Furigana on card backs
- Audio playback using the original online audio URLs
- Local browser progress storage
- `J` marks known, `F` marks missed, Space flips the card

## Data Note

This site is intended for learners who own the corresponding vocabulary material.
